# Hi, Stranger

A simple, elegant homepage with a soft butter yellow background and chocolate brown text.

## Features

- Centered "Hi, stranger" greeting in Bodoni serif font
- Fully rounded "Tell me a secret" button
- Soft color palette with butter yellow background (#F4E4C1) and chocolate brown text (#6B4423)
- Responsive design that works on all screen sizes
- Smooth button interactions with hover and active states

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and layout

## Usage

Open `index.html` in your web browser to view the homepage.
